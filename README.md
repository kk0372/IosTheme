# SillyTavern Apple Dark UI Theme

A modern, Apple-inspired glassmorphism theme extension for SillyTavern.

## ✨ Features
- **Apple Design Language**: Clean typography, rounded corners, and vibrant accent colors.
- **Glassmorphism**: Beautiful frosted glass effects using backdrop filters.
- **Quick Actions**: Custom floating buttons for Reroll, TTS, Translation, and more.
- **Dynamic Adjustments**: Customize UI via the in-app extension menu.

## 📦 Installation via GitHub
1. Open SillyTavern and go to the **Extensions** menu (plug icon).
2. Click **Install Extension** and select **Install from GitHub**.
3. Paste the URL of this repository.
4. Click Connect and restart SillyTavern.

## ⚙️ Usage
Once installed, a new menu called "🍎 Apple Dark UI Settings" will appear in your Extensions panel. You can adjust:
- Accent colors and glow effects.
- Backdrop blur intensity and opacity.
- Border radius and avatar shapes.
